# Les contrastes
Les contrastes peuvent varier en fonction de la taille du texte et de son affichage en gras ou non

- Contrast 4.5:1
  - < 24px
  - < 18.5px gras
<br><br>

- Contrast 3:1
  - \>= 24px
  - \>= 18.5px gras


Notes:
Voir RGAA pour être précis mais dans l'idée:
> 24 non gras : 4.5:1
> 18.5 gras : 4.5:1
<= 24 non gras : 3:1
<= 18.5 gras : 3:1
