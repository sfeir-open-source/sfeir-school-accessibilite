## Develop

To run docs locally, open your favorite terminal, go in directory `docs`, install dependencies using npm and run `npm run start`, then open slides on http://localhost:4242/.

## Workshop

Workshops are in directory `steps` :
* The main exercices will be to make accessible a given website
* Each exercises are in the corresponding step
* The correction will be in the next exercise
